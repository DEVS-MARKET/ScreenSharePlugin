Plugin for Minecraft 1.21 and Java 21, to create a screen that shares your current image in Minecraft, this is a recode of the project created by [MystiCat](https://www.youtube.com/watch?v=mblXXy5hjM0) but posted completely for free! üî•

## Basic information

This plugin only works on the latest versions of Minecraft 1.21 due to the entity attribute added in these versions, it is also recommended to use Java 21. This plugin only works for localhost and servers with a graphical interface!
